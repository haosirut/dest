# Оферта для клиентов VaultKeeper

**Версия:** 1.0 | **Дата:** 2024-12-01

---

## 1. Термины и определения

| Термин | Определение |
|--------|-------------|
| **Платформа** | Программное обеспечение VaultKeeper, включая серверную и клиентскую части |
| **Клиент** | Физическое лицо, использующее Платформу для хранения данных |
| **Хранитель** | Физическое лицо (самозанятый/ИП), предоставляющее дисковое пространство Сети |
| **Сеть** | Децентрализованная P2P-сеть VaultKeeper |
| **Чанк** | Зашифрованный фрагмент данных размером 4 МБ |
| **Реплика** | Копия чанка, хранимая у разных Хранителей |

## 2. Предмет оферты

Платформа предоставляет Клиенту услуги по хранению данных в децентрализованной P2P-сети. Все данные шифруются на устройстве Клиента до передачи в Сеть. Платформа не имеет доступа к содержимому хранимых данных.

## 3. Агентская схема расчётов

Платформа действует как **агент Хранителя** в соответствии со ст. 1005 ГК РФ:

- **90% платежа** — вознаграждение Хранителю (принципалу) за предоставление дискового пространства
- **10% платежа** — комиссия Платформы (агента) за техническое обеспечение, маршрутизацию и проверку целостности данных

Налоговые обязательства:
- Налог с 10% (комиссия агента) уплачивается Платформой самостоятельно
- Налог с 90% (вознаграждение Хранителя) уплачивается Хранителем самостоятельно в рамках НПД/ИП

## 4. Тарифы

| Параметр | Значение |
|----------|----------|
| Базовая ставка | 0,30 ₽/ТБ/час |
| Множитель HDD | ×0,80 |
| Множитель SSD | ×1,00 |
| Множитель NVMe | ×1,50 |
| Мин. пополнение баланса | 100 ₽ |
| Заморозка при балансе | ≤ 0 ₽ |

## 5. Шифрование

- Алгоритм: XChaCha20-Poly1305 (AEAD)
- Ключ derives: Argon2id (память 64 МБ, итерации 3)
- Seed-фраза: BIP39 (12 слов)
- Erasure coding: Reed-Solomon (3 из 5 чанков)

## 6. Ответственность

Платформа не несёт ответственности за:
- Утрату данных, если менее 3 из 5 реплик доступны
- Доступ третьих лиц к seed-фразе Клиента
- Принудительное изъятие данных у Хранителей

## 7. Расторжение

Клиент может расторгнуть оферту в любой момент. Возврат средств за неиспользованный баланс осуществляется в течение 30 дней.

---

*Акцептом оферты является создание учётной записи в приложении VaultKeeper.*

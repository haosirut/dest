## 3. КАТЕГОРИИ СОБИРАЕМЫХ И ОБРАБАТЫВАЕМЫХ ДАННЫХ

3.1. При регистрации учётной записи Платформа обрабатывает следующие персональные данные:

- **Идентификационные данные:** адрес электронной почты, номер телефона (при указании), отображаемое имя пользователя (псевдоним).
- **Данные для идентификации Хостов:** полное имя, данные паспорта, ИНН, СНИЛС — только для Хостов, заключающих Договор хранения. Эти данные обрабатываются в целях выполнения налогового и финансового законодательства.
- **Технические данные:** IP-адрес, тип и версия операционной системы, идентификатор устройства, данные браузера/приложения, логи подключений.
- **Финансовые данные:** история платежей, реквизиты банковского счёта (для Хостов). Номера банковских карт не сохраняются на серверах Платформы — обработка осуществляется через платёжных партнёров (YooKassa) в соответствии с стандартом PCI DSS.
- **Метаданные хранилища:** объём использованного хранилища, количество файлов, хеш-суммы, временные метки операций.

3.2. **Данные, которые Платформа НЕ обрабатывает:**

- Содержимое пользовательских файлов. Все данные шифруются на стороне клиента (client-side encryption). Ключи шифрования хранятся исключительно на устройстве Пользователя. Платформа технически не имеет возможности доступа к содержимому файлов.
- Пароли учётных записей (хранятся в виде криптографических хешей с использованием bcrypt/argon2).

7.1. Персональные данные хранятся в течение срока, необходимого для достижения целей их обработки, но не менее сроков, установленных законодательством:

| Категория данных                     | Срок хранения                  |
|-------------------------------------|--------------------------------|
| Данные учётной записи               | Весь период использования Сервиса + 3 года после удаления аккаунта |

| Метаданные хранилища                | Период хранения данных + 30 дней |
| Данные cookie                       | До удаления Пользователем или истечения срока действия |

7.2. По истечении сроков хранения персональные данные уничтожаются путём необратимого удаления (secure erase) из всех информационных систем Оператора.

## 9. ФАЙЛЫ COOKIE И ТЕХНОЛОГИИ ТРЕКИНГА

9.1. Сервис использует файлы cookie и аналогичные технологии для:

- Аутентификации Пользователя (session cookies).
- Запоминания настроек и предпочтений.
- Аналитики использования Сервиса (анонимизированные данные).
- Обеспечения безопасности (защита от CSRF-атак).

9.2. Сервис использует следующие категории cookie:

| Категория        | Назначение                     | Срок            | Требует согласия |
|-----------------|--------------------------------|-----------------|------------------|
| Необходимые     | Аутентификация, безопасность    | До закрытия сессии | Нет            |
| Функциональные  | Настройки языка, темы           | 1 год          | Да              |
| Аналитические   | Статистика использования        | 2 года         | Да              |

9.3. Пользователь может управлять файлами cookie через настройки браузера. Отключение необходимых cookie может привести к невозможности использования Сервиса.
